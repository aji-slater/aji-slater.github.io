Sublime Text 3

Keyboard Shortcuts

EDITING
cmd + shfit + up move line/selection up
cmd + shift + down move line/selection down
cmd + D select word, repeat to select next instances
cmd + m jump to closing parenthesis, repeat to jump to opening
cmd + shift + m Select everything within current parenthesis
cmd + shift + K delete current line
cmd + KK delete from cursor to end of line
cmd + k + backspace delete from cursor to beginning of line
cmd + ]/[ indent/unindent current line
cmd + j join line below to end of current line
cmd + / toggle comment of current line/selection
cmd + shift + v paste with corrent indentation
cmd + click duplicate cursor to click
WINDOWS & TABS
cmd + shift + T open last closed tab
alt + (num) switch to (num) tab
alt + shift + 1 set view to one column
alt +shift + 2 set view to 2 columns
